HTML

  • HTML mean Hypertext Markup Language.
  • We use HTML to give content and structure to website.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The meta element contains information for the browser.
  • ul element means unordered list while ol element means ordered(1,2,3 and etc)

CSS

  • CSS stands for Cascading Style Sheets.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Each element is like a box, in programming it is called box model.
  • CSS is used to add style for your application.
  • There are 3 types of CSS: inline,internal and external.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JavaScript allows you to make web pages interactive.
  • Loops are used in JavaScript to perform repeated tasks based on a condition.
  • JavaScript Array is a single variable that is used to store elements of different data types.
  • A function in JavaScript is similar to a procedure—a set of statements that performs a task or calculates a value.
  • A function is declared using the function keyword.
